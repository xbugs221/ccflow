# 提案：优化项目主页会话卡片展示

## 目标

改进项目主页会话卡片的用户体验，使其与左侧导航栏保持一致的时间展示格式，并增加状态指示功能。

## 非目标

- 不修改后端 API 或数据模型
- 不引入服务端已读/未读状态同步
- 不影响现有侧边栏行为

## 变更范围

1. **时间戳格式**：项目主页会话卡片的时间戳从绝对时间（`2026-04-28 16:07`）改为距今时间（`5 mins ago`）
2. **未读状态指示灯**：在会话卡片右上角添加黄色圆点，区分已读和未读消息
3. **右键菜单文字**：会话卡片右键菜单从纯图标改为图标+文字描述

## 参考实现

- 左侧导航栏 `SidebarSessionItem.tsx` 已使用 `formatTimeAgo` 和未读状态灯
- 右键菜单组件 `SessionActionIconMenu.tsx` 当前为 icon-only 设计
